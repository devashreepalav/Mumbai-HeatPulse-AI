# 🏙️ Mumbai HeatPulse: AI Urban Climate Analysis

A live AI-driven dashboard analyzing the **Urban Heat Island (UHI)** effect across Mumbai's wards. This project correlates vegetation density (NDVI), building density, and socioeconomic factors to predict and visualize surface temperature variances.

[Live Dashboard Link Here]

## 🚀 Key Features
* **Live Satellite Integration:** Fetches real-time temperature data via Open-Meteo API.
* **Interactive Ward Analysis:** Visualizes the relationship between "Greenery vs. Heat" using Plotly.
* **Socioeconomic Correlation:** Identifies how income levels and building density impact localized climate.
* **AI Insights:** Automated logic to identify the temperature gap between forested and concrete-heavy wards.

## 🛠️ Tech Stack
* **Language:** Python 3.x
* **Interface:** Streamlit
* **Data Science:** Pandas, Scikit-Learn
* **Visualization:** Plotly Express
* **Deployment:** GitHub & Streamlit Community Cloud

## 📊 Data Insights
Our analysis reveals a **4.2°C temperature variance** between high-vegetation wards (like R/C) and low-vegetation, high-density wards (like G/N). This highlights the critical need for urban forestry in Mumbai's development.
